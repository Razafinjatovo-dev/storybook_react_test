# README #

Storybook test

### How do I get set up? ###

* Clone the repo
* `npm install`
* `npm run storybook`

Please commit on a new branch a make a pull request.
Add you css style in `./stories/assets/scss/dom-tom.scss` and the new components 
in `./stories/src/mobile/components/`
